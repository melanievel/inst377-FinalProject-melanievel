You will first start at the Home Page

You'll see the Home screen and at the bottom there is a button that you can click to generate a random recipe! When you click the button, you will be redirected to another page with the details of the random recipe that was generated: image, title, description, and instructions. At the bottom of this page, you can click the 'check out other recipes!' and it will redirect you to the recipes page.

At the Recipes Page

To the left, there is a form that users can fill out if they want to search for recipes with specific categories such as: diet, intolerance, and cuisine. When the form is submitted, a list of recipes that meet all of the chosen categories will be displayed. Once the user looks through the recipe results, they can copy the id of a recipe they liked and paste it in the search bar to the right. When they click submit, they'll be able to see more details about the recipe such as a full link to the recipe source, instructions, and more!

The About Page

This page will simply give the user some insight on the purpose of the web page; the issue, why the web page was created, and why the web page supports the issue.

Navigation

There's also a navigation bar visible in every page to allow for easy use and navigation within the web page. Users can simply click the tab with the name of the page they wish to visit, and they will be redirected to the desired page.

You can navigate this webpage using audio!

Ensure your mic is enabled to be able to use this feature

  • Navigate to a different page
    - Say: Navigate to <page>
  • Change the page color
    - Say: Change the color to <color>
